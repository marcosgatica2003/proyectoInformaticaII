Descripción detallada

DHT11.h Header file for the DHT11 library, providing functionalities to interface with the DHT11 temperature & humidity sensor.

Author: Dhruba Saha Version: 2.1.0 License: MIT DHT11 Class Provides methods to read temperature and humidity data from the DHT11 sensor.
